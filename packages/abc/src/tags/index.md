简化标签参数传递，常用于状态显示。


## API

### 属性

| 成员 | 说明 | 类型 | 默认值 |
|----|----|----|-----|
| `tagMap` | 文本字典 ｜ `ITag` | - |
| `value` | 值 | `any` | - | 

### tagMap

| 成员 | 说明 | 类型 | 默认值 |
|----|----|----|-----|
| `color` | 标签颜色 | `string` | - |
| `text` | 显示文本 | `string` | - |
