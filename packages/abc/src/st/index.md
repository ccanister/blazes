:::content
`st` 并不是在创造另一个表格组件，而是在 `a-table` 基础上以**可配置**形式渲染表格，在中后台里这种方式可以满足绝大多数场景，但又可以更易地管理表格渲染动作。
:::

## API

### 属性

| 属性        | 说明   | 类型          | 默认值 |
| ----------- | ------ | ------------- | ------ |
| `[columns]` | 列描述 | `ISTColumn[]` | -      |
