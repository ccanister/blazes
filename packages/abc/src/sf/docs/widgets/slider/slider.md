滑动型输入器，展示当前值和可选范围。

## API

### schema 属性

| 成员           | 说明   | 类型     | 默认值 |
| -------------- | ------ | -------- | ------ |
| `[minimum]`    | 最小值 | `number` | `0`    |
| `[maximum]`    | 最大值 | `number` | `100`  |
| `[multipleOf]` | 倍数   | `number` | `1`    |

### ui 属性

| 成员       | 说明                           | 类型                               | 默认值 |
| ---------- | ------------------------------ | ---------------------------------- | ------ |
| `[range]`  | 当添加该属性时，启动双滑块模式 | `Boolean`                          | -      |
| `[change]` | 滑动值回调函数                 | `(value:number \| number[])=>void` | -      |
