一般用于多行字符串。

## API

### schema 属性

| 成员          | 说明         | 类型      | 默认值 |
| ------------- | ------------ | --------- | ------ |
| `[maxLength]` | 表单最大长度 | `number`  | -      |
| `[readOnly]`  | 禁用状态     | `boolean` | -      |

### ui 属性

| 成员            | 说明                                                                    | 类型                    | 默认值 |
| --------------- | ----------------------------------------------------------------------- | ----------------------- | ------ |
| `[placeholder]` | 在文字框中显示提示讯息                                                  | `string`                | -      |
| `[autosize]`    | 自适应内容高度，可设置为 `boolean` 或对象：`{ minRows: 2, maxRows: 6 }` | `Boolean \| Object`     | `true` |
| `[change]`      | 内容变更事件                                                            | `(val: string) => void` | -      |
