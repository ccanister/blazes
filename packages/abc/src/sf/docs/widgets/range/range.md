范围日期选择控件。

## API

### schema 属性

| 成员         | 说明     | 类型      | 默认值 |
| ------------ | -------- | --------- | ------ |
| `[readOnly]` | 禁用状态 | `boolean` | -      |

### ui 属性

| 参数             | 说明               | 类型                         | 默认值 |
| ---------------- | ------------------ | ---------------------------- | ------ |
| `[separator]`    | 分隔符             | `string`                     | -      |
| `[disabledDate]` | 不可选择的日期     | `(current: Date) => boolean` | -      |
| `[change]`       | 时间发生变化的回调 | `(data: Date) => void`       | -      |
| `[format]`       | 数据格式类型       | `string`                     | -      |
